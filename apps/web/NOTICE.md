# Web design notice

MindTrain's Web design tokens and visual language are adapted from
[shigella520/LinkPeek](https://github.com/shigella520/LinkPeek), which is available under the MIT License.

The MindTrain Web application uses Vue 3, TypeScript, Vite, Vue Router and Pinia. LinkPeek's source CSS is not bundled directly; MindTrain reimplements the visual language for its own Dashboard, training and administration workflows.
