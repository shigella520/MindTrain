# Source Policy

Prefer Java/JLS/JEP/OpenJDK documentation, then official framework, database, and middleware documentation, then established technical publications. Use interview-experience posts only to discover topics.

Reuse sources already linked by the selected taxonomy topic. Search the web only when stored sources are insufficient, version-sensitive, or likely outdated. Record source ID, URL, title, access date, and applicable version.

Save every live AI question under `assets/candidates/` with `status: candidate`. It may be used in the current session after validation, but must not be reused or moved to `assets/questions/` until source review, duplicate checking, and human approval are complete.
